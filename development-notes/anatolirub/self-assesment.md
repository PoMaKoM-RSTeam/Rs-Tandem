## 2. Описание проделанной работы

В рамках проекта я реализовывал функциональность регистрации, аутентификации и авторизации. Написал для этого необходимые страницы и компоненты, использовал ленивую загрузку для них. Для частей регистрации и аутентификации использовался supabse-js и поверх него api layer c котрым собственно и работает приложение.
Также я реализовывал not-found страницу при переходе на незнакомый приложению роут(**wildcat).
Также реализовывался ai-agent помощник, который представляет собой RAG систему. Что было реализовано: чтение книг в txt и pdf, разбиение их на чанки, конвертация чанков векторное представление, сохранение векторных представлений в супабэйз. Часть которая отвечает за ответы включает себя гибридный поиск и реранкер чанков для предоставления наиболее точных частей, которые будут составлять контекст для модели генерирующей ответ, это позволяет избавиться от галюцинаций модели и базировать свои ответы на проверенной информации.
