## 2. Описание проделанной работы

В ходе финального этапа обучения я с нуля реализовал два ключевых игровых модуля: "Code Golf" (соревнование по написанию минималистичного кода) и "Sandbox" (интерактивная среда для фронтенд-разработки). Работа велась в команде из 5 человек с использованием гибких методологий (Scrum-созвоны, GitHub Issues, Code Review). Проект научил меня работать в условиях неопределенности и свободы выбора. Я не только освоил Angular на практике, но и научился воспринимать AI как эффективного помощника в рутинных задачах. Самым ценным опытом стало взаимодействие с командой и ментором Аней — это помогло увидеть архитектурные ошибки, которые сложно заметить при одиночной разработке.


## 3. Два личных Feature Component (для защиты)

В качестве компонентов, разработанных мной лично, я хочу выделить следующие:

### 1. Complex Component `Code Golf`

Code Golf представляет собой тренажер для развития навыков оптимизации кода, где основной акцент сделан на написании максимально лаконичных и эффективных алгоритмов. В основе архитектуры модуля лежат современные возможности Angular Signals, которые обеспечивают мгновенную реактивную обратную связь: система в реальном времени анализирует объем решения, очищает его от избыточных символов и визуализирует текущий прогресс пользователя. Особое внимание уделено безопасности и стабильности интерфейса, для чего была реализована изоляция исполнения пользовательского кода через Web Workers. Это решение позволяет тестировать алгоритмы в отдельном потоке, предотвращая блокировку основного UI и гарантируя защиту приложения от бесконечных циклов или критических ошибок в решениях. Глубокая интеграция с Supabase превращает процесс решения задач в полноценную систему личных рекордов, позволяя пользователям отслеживать свой рост в реальном времени через сравнение текущих результатов с предыдущими лучшими показателями. Такой подход сохранения данных в сочетании с динамическим расчетом веса кода создает мотивирующую среду для непрерывного совершенствования навыков программирования.

### 2. Complex Component `Sandbox`

The Sandbox представляет собой интерактивную среду для фронтенд-разработки, спроектированную на базе технологий Monaco Editor и Angular Signals. В основу архитектуры модуля заложен паттерн Facade, который позволил достичь исключительной чистоты кода за счет выноса всей сложной логики управления состоянием и взаимодействия с API в отдельный сервисный слой. Это решение обеспечивает высокую производительность Live Preview: система реактивно отслеживает изменения в HTML, CSS и JavaScript, мгновенно обновляя результат без лишних перерисовок интерфейса.

Особое внимание в разработке уделено стабильности приложения. Для рендеринга пользовательского кода используется динамически генерируемый изолированный iframe, что полностью исключает влияние стилей или скриптов из «песочницы» на работу основного приложения. Помимо инструментов для экспериментов, в модуль интегрирована полноценная система хранения данных через Supabase. Пользователи могут сохранять свои наработки в облачную базу данных и в любой момент возвращаться к редактированию ранее созданного кода.
